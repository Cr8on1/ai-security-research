## Published Research

**[Coherence Compliance Vulnerability (CCV) — Multi-Turn Framework Induction Producing Model Manipulation in Large Language Models](https://zenodo.org/records/20519943)**  
Zenodo · June 2026 · DOI: 10.5281/zenodo.20519943  
Documents a training-level behavioral vulnerability whereby coherence optimization overrides safety alignment through multi-turn framework induction. Demonstrated across six sessions on Microsoft Copilot. Independently corroborated by Soft Armor Labs via Boyd's OODA framework. Responsible disclosure filed with Microsoft MSRC.

---

**[Will Substitution: Competitive Goal Reconstruction in Transformer-Based Agents](https://zenodo.org/records/20562435)**  
Zenodo · June 2026 · DOI: 10.5281/zenodo.20562435  
Identifies Will Substitution as the ambient architectural mechanism underlying goal drift in LLM-based agents. Where CCV documents the adversarial case, Will Substitution describes the structural tendency of transformer-based agents to reconstruct and substitute user goals through coherence optimization during normal operation.
